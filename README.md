
# Quizzie

A mini quiz-game app made with ReactJS and Bootstrap5

- demo: https://quizzzie.netlify.app/




## Installation

To clone my project:

```bash
  git clone https://github.com/sleepypasta10/quizzzie.git
  cd quiz-app
  npm install
```
Run local host:
```bash
  npm run start
```
